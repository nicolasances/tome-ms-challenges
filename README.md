# Tome Challenges

This microservice provides APIs to manage Tome Challenges.

The core aspects of Challenges are the following:

* A Challenge starts with a **Context**. <br>
    The Context helps the User understand the Challenge and the questions that will come. It places the questions in a context. <br>
    An example of Context, for a historical challenge, is to provide a high level view of the historical period the challenge is related to.
